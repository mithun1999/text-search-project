# Text Search Project

## Tech Stack (Typescript)
- Client - React
- Server - NestJS
- DB - MongoDB
- Cache - Redis

## Installation

Install the dependencies and devDependencies and start the server. Before starting the server, have your `redis-server` running in background. 

```sh
yarn
yarn start:dev
```

To run the client,

```sh
yarn
yarn dev
```
